Novo Launcher - Minecraft 启动器功能介绍

【核心功能】
1. 游戏下载：支持下载 Minecraft 各版本（正式版、快照版、远古版本），使用 BMCLAPI 国内镜像加速下载。

2. 游戏启动：自动配置 Java 环境，支持自定义 JVM 参数，一键启动游戏。

3. 版本隔离：每个游戏版本拥有独立的存档、截图、资源包目录，互不干扰。

4. 账户管理：支持离线模式和微软正版登录，可管理多个账户。

5. 存档管理：自动追踪各版本存档，支持存档详情查看、备份与删除。

6. Mod 管理：集成 CurseForge 模组搜索，支持按版本和加载器筛选，一键下载安装。

7. 主题切换：支持亮色/暗色主题，提供流畅的动画效果。

8. 下载管理：支持断点续传、多镜像源切换、下载进度实时显示。

【技术特点】
- 使用 .NET 8 + WPF 开发
- 支持 Windows 系统
- 自动解压 Natives 库文件
- 完整的错误日志记录
