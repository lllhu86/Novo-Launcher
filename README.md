



#  Novo Launcher 功能介绍

---

## 一、游戏下载与版本管理

打开启动器，点一下「加载版本列表」，你就能看到从远古 Alpha 到最新快照的全部 Minecraft 版本。我特意把它们分成了 **「正式版」「预览版」「远古版」「愚人节版」** 四个车厢，找起来一目了然。

下载内置了 **BMCLAPI 国内镜像**，速度起飞。断点续传、多线程加速、校验失败自动重试，全套安排。你只管点 开始游戏。


---

## 二、账号系统 —— 正版离线，自由切换

支持 **微软正版登录** 和 **离线账号**。微软登录走的是 OAuth 设备码流程，你只需要在浏览器里输入代码，剩下的 Xbox Live、XSTS、Minecraft 三层认证我全自动搞定。Token 还用 DPAPI 加密保存在本地，安全得很。

离线账号随便起名，UUID 自动生成。

---

## 三、存档追踪系统 

Novo Launcher 会在你玩游戏时默默记录，**记录但几乎不占性能**：

-  **记录游玩时长**：精确到秒，知道你在这个世界投入了多少。
-  **记录最后坐标、维度、游戏模式、难度、天气、种子**：从主世界跳到下界，从生存切到创造，全记上。
-  **自动生成「最近存档」卡片**：主页就能看到「xx 小时前玩过的世界」，双击直接启动，不用翻文件夹。

它甚至能实时监控 `level.dat` 的变化

---

## 四、Mod 管理 —— 模组市场开在启动器里

集成 **Modrinth API**，搜索、筛选（Forge/Fabric）、查看详情、下载、安装一条龙。我还加了个翻译服务，把英文描述机翻成中文（虽然偶尔翻得有点憨，但比看天书强 qwq）。

**依赖分析**：你下一个 Mod，我能告诉你它需要哪些前置

---

## 五、整合包支持

支持导入 `.mrpack` 和 CurseForge 整合包，也能在线浏览热门整合包直接下载。导进来之后自动解压、创建实例、分配独立目录。你只管点「启动游戏」，整合包环境我帮你搭好帕。

>  **画饼时间**：未来我还打算做一个「整合包分享」功能，你调好的配置可以一键导出给朋友。

---

## 六、AI 助手「小矿工」 

这是 Novo Launcher 的 **灵魂** ！

「小矿工」不是一个简单的聊天框，它是一套完整的 **游戏智能系统**：

-  **上下文感知**：它知道你的 CPU 型号、内存大小、显卡型号，知道你已经安装了哪些版本、有哪些存档。回答问题是「对症下药」而不是「套话模板」。
-  **实时日志监控**：游戏一崩溃，它立刻弹窗告诉你：「检测到 Exit Code 1，疑似内存不足，建议把分配内存从 2G 调到 4G」——并提供一个 **「一键应用」** 按钮。
-  **主动诊断**：点一下「问题诊断」，它自动扫描你的 Mod 冲突、崩溃报告、启动器日志，生成一份傻瓜都能看懂的诊断报告。
-  **资源搜索与安装**：你对它说「帮我找个 1.20 可用的光影」，它直接去 Modrinth 搜，列出来让你选编号，选完自动下载到正确文件夹。
-  **主动提醒**：它会在后台悄悄检查——你多久没备份存档了？你电脑内存是不是太小了？OptiFine 和 Sodium 是不是打架了？然后像老妈子一样弹窗提醒你帕。

**这不是一个「附加功能」，这是 Novo Launcher 的「第二心脏」。**

## 八、主题与动画

深色/浅色主题一键切换，切换时还有淡入淡出动画。按钮悬停会微微放大，按下会轻轻缩小，列表项加载时从左侧滑入……这些「没用但爽」的细节

---

## 九、未来开发计划

###  服务器列表功能（v2.0 计划）

我想做一个内置的服务器浏览器，让你们不用记 IP 地址，直接在里面看在线人数、延迟、版本，点一下就能加入。开发者可以提交自己的服务器信息，玩家可以收藏常玩的服。目标是做成 **「Minecraft 服务器的大众点评」** 。

### ☁️ 云存档功能（v2.2 计划）

这是我最想做的功能！你们有没有过这样的痛：在家里的电脑上玩了一个存档，到了学校/公司想接着玩，只能 U 盘拷来拷去，最后版本还乱了。

云存档要解决的就是这个。**自动上传、多端同步、冲突合并、增量备份**。普通用户送 5GB 空间，捐赠的大佬送 50GB。**让你的世界跟着你走，而不是困在一台电脑里。**
